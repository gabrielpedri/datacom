//Acesso Serial ou Remoto - CLI

//Command Line Interface
//Meio utilizado para configurar o DmOS de forma local através da porta console/serial ou remota por SSH

//O endereli IP padrão para acesso é o 192.168.0.25/24
//Por default, o usuário com privilégios de configuração é o admin com senha admin

//Habilitar telnet
DmOS(config)#telnet-server
DmOS(config)#commit

//Para suporte a conectividade com equipamentos que utilizam OpenSSH com versão inferior a 7.0
DmOS(config)#ssh-server legacy-support
DmOS(config)#commit

//Recursos Utilizaveis

?           --> Usado para apresentar lista de comandos que completará o comando
Tab         --> Completa o comando
Abreviações --> Permite inserir comando de forma abreviada
do          --> Permite executar comandos fora do seu menu de execução
^           --> Sinaliza onde há erro de sintaxe
